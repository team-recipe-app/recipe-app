# Recipe App

## 概要
食材から作れる料理を提案するアプリ

---

## 機能
- 食材入力  
- レシピ提案  
- 買い物リスト作成  

---

## 使用技術

### 開発言語
- フロントエンド：HTML / CSS / JavaScript  
- バックエンド：Python  

### フレームワーク
- Django  
- Node.js（サブ）

### データベース
- MySQL  

### インフラ
- AWS  
- Firebase  

### API
- OpenAI API  
- Rakuten Developers  

### ツール
- VSCode  
- GitHub  
- Docker  

---

## 目的
料理初心者でも簡単に料理を選べるようにする
